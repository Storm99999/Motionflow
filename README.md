# Motionflow
A performant and compact Animation engine, written in LuaU

# Setup
1. Get the [model](https://create.roblox.com/store/asset/126182263894495/Motionflow-10) or the [Test place](https://github.com/Storm99999/Motionflow/releases/download/Main/MotionflowExample.rbxl)
2. Place the "MotionCore" LocalScript into StarterPlayerScripts, or use a custom module loader.
3. Profit
